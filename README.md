# Monitoring-door-WIFI
Using Arduino Uno and NodeMCU LoLin v3 ESP8266
- Magnetic reed sensor (normally open)
- 10kΩ Resistor
- Bidirectional Logic Level Converter BOB-12009 3.3 to 5V
